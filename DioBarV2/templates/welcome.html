Welcome to RebarVista

This application allows you to detect and analyze rebar sections, calculate their volume, and measure distances. The system uses advanced computer vision to help you work more efficiently.

Live Camera Feed

View real-time footage from your camera with distance measurements

Distance Measurement

Measure distances using the ultrasonic sensor

Volume Calculation

Automatically calculate the volume of detected rebar sections

Export Results

Download results as images, CSV, or PDF

How to Get Started

  1. Ensure your Raspberry Pi is connected and the camera is properly set up
  2. Connect to the same network as your Raspberry Pi
  3. Click the Start button below to begin using the application
  4. Use the capture button to analyze rebar in the camera view

Tip: For optimal results, ensure good lighting and keep the camera steady.
